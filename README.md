# CODERIUM
### Team Project(GUI)

- 기획의도<br>
  - 예약 기능이 있는 스터디카페 키오스크 구현

- 개발인원(책임파트)<br>
  - 나문오[팀장](로그인, 회원가입), 정원호(좌석), 최민규(예약), 오지수(결제)

- 개발환경<br>
  - Platform : Windows 10 x64, i7-8565U, 16G / macOS Big Sur, i5, 8G<br>
  - Language : Java(JDK 1.8), MySQL<br>
  - Dev Tool : Eclipse 4.17<br>
  - VCS : Git<br>
